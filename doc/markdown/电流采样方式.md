# 2 低侧采样说明
![](Pasted%20image%2020260617210824.png)

+ 可能是最常见的电流检测技术
+ 优点
	+ 不需要高性能的PWM抑制运放（如内置）
	+ 不需要运放支持高压
+ 缺点：
	+ 必须在下桥臂MOS打开时检测电流，也就是：必须在电流流过采样电阻时去采样
	+ 需要避开开关噪声，在MOS开关切换之后的稳定区去采样
		+ PWM频率通常为20k～50khz
		+ 这意味着低侧MOS的开关频率为每秒20k～50k次

		+ 最好是用那种支持硬件触发采样的MCU
+ 将ADC设置成用timer触发，到达合适时机后进行采样
