# 2 采样方式详细说明
## 2.1 低侧采样说明
![](attachments/lowside.png)
+ **LowSideCurrentSense**
+ **可能是最常见的电流检测技术**
+ 优点
	+ 对运放的要求会比较低
		+ 比如不需要运放支持很高的电压，因为低侧的电压不高
		+ 比如对运放共模抑制比的要求可以弱一点
+ 缺单
	+ 需求期间采不到
	+ 地线耦合噪声
<br>

## 2.2 高侧采样说明
![](attachments/highside.png)
+ **HighSideCurrentSense**
+ **可能是最不常见的电流检测技术**
+ 优点
	+ 可检测短路到地的故障
	+ 不受地线噪声影响
+ 缺点
	+ 运放支持的电压需要比较高
		+ 具体多高取决于母线电压
	+ 需要运放有较好的共模抑制比
		+ 否则共模泄露会严重影响电流采样
<br>

## 2.3 **相线串联采样**
![](attachments/inline.png)
+ **InlineCurrentSense**
+ **实时性能最好**
+ 优点
	+ 直接测相电流，不存在盲区
		+ 不需要像高侧或者低侧那样，卡在在桥臂导通的时机去采集
+ 缺点
	+ 相线上一直有电流，增加功耗，对采样电阻的要求提高
	+ 有布局上的要求，否则相电流不平衡（这个没有很懂，回头再细究）
	+ Inline 电阻增加相线电感，高速开关时振铃更大（不是很懂）
<br>

## 2.4 单电阻母线采样
+ 高侧单电阻：电源->电阻->三个上桥臂
+ 低侧单电阻：三个上桥臂->电阻->GND
<br>

# 3 注意点
## 3.1 高侧、低侧采样共通点
+ 必须在桥臂MOS打开时检测电流
	+ 也就是：必须在电流流过采样电阻时去采样
+ 需要避开开关噪声，在MOS开关切换之后的稳定区去采样
	+ MOS切换时会有振铃之类的干扰信号
+ 运放需要能够快速响应电流变化
	+ PWM频率通常为20k～50khz
	+ 运放需要在电流导通时，快速简历，稳定输出采样值
+ 可以考虑使用支持硬件触发采样的MCU
+ 将ADC设置成用timer触发，到达合适时机后进行采样
<br>

# 4 适用场景

## 4.1 低侧采样

| 控制方式     | 适用性      | 场景                                     |
| :------- | :------- | :------------------------------------- |
| **BLDC** | ✅ 适合     | 六步换相，成本敏感，只需知道当前导通相电流                  |
| **FOC**  | ✅ **主流** | 双电阻或三电阻，直接测相电流， Clarke/Park 变换需要 Ia、Ib |
**BLDC 用法**：单电阻或双电阻在下桥臂，换相时采对应相。 **FOC 用法**：双电阻（U、V 相）最常用，第三相计算得出。
<br>

## 4.2 高侧采样

| 控制方式     | 适用性       | 场景                 |
| :------- | :-------- | :----------------- |
| **BLDC** | ⚠️ 可用但少见  | 需要检测短路到地故障，汽车级安全要求 |
| **FOC**  | ⚠️ 可用但成本高 | 工业伺服、汽车电机，预算充足     |

**特点**：共模电压高，需专用芯片（INA240），成本是低侧的 3-5 倍。
<br>

## 4.3 相线串联采样（Inline）

| 控制方式     | 适用性      | 场景                       |
| :------- | :------- | :----------------------- |
| **BLDC** | ❌ 浪费     | 六步换相只需单电阻，Inline 成本高无优势  |
| **FOC**  | ✅ **最佳** | 高性能伺服、无人机、无传感器 FOC，无采样盲区 |

**特点**：每 PWM 周期都能采两相电流，带宽最高，但电阻功率和 PCB 要求高。
<br>

## 4.4 高/低侧单电阻（Single Shunt）

|控制方式|适用性|场景|
|:--|:--|:--|
|**BLDC**|✅ **最常用**|低成本电调、风扇、水泵，六步换相够用|
|**FOC**|⚠️ 可用但受限|简单 FOC、成本极度敏感，需算法重构，有盲区|

**特点**：1 个电阻 + 1 个运放，成本最低。FOC 时需 SVPWM 移相或特殊采样窗口。
<br>
