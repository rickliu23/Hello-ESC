
# 1 运放ADI OP291，测试用

## 1.1 电源特性

| 参数         | 数值                                |
| :--------- | :-------------------------------- |
| 供电范围       | **2.7V ~ 12V**（单电源）或 **±5V**（双电源） |
| 静态电流       | **300 μA/通道**（典型值）                |
| 电源抑制比 PSRR | 80 ~ 110 dB                       |


## 1.2 输入特性

| 参数         | 数值                        | 说明                                                                                    |
| :--------- | :------------------------ | ------------------------------------------------------------------------------------- |
| 输入失调电压     | **80 μV（典型）/ 700 μV（最大）** | **理想运放**：当输入差分电压为 0 时，输出应为 0。<br><br>**实际运放**：由于内部晶体管不完全对称，需要施加一个**微小的差分电压**才能使输出为 0。 |
| 输入失调电压漂移   | **1.1 μV/°C**             |                                                                                       |
| 输入偏置电流     | **30 nA（典型）/ 65 nA（最大）**  |                                                                                       |
| 输入电压范围     | **轨到轨输入**（可超出电源轨 10V）     |                                                                                       |
| 共模抑制比 CMRR | 70 ~ 93 dB                |                                                                                       |

## 1.3 输出特性

| 参数     | 数值                      |
| :----- | :---------------------- |
| 输出电压摆幅 | **轨到轨输出**（距轨约 10~50 mV） |
| 输出短路电流 | ±8.75 ~ ±13.5 mA        |
| 负载驱动   | 可驱动 2kΩ 至轨附近            |

## 1.4 动态性能

| 参数        | 数值                                     |
| :-------- | :------------------------------------- |
| 增益带宽积 GBW | **3 MHz**                              |
| 压摆率 SR    | **0.4 V/μs**（3V 供电）/ **0.5 V/μs**（±5V） |
| 建立时间      | **22 μs**（到 0.01%）                     |
| 相位裕度      | 45°                                    |

## 1.5 噪声性能

| 参数             | 数值                                   |
| :------------- | :----------------------------------- |
| 电压噪声（0.1~10Hz） | **2 μV p-p**                         |
| 电压噪声密度（1kHz）   | **30 nV/√Hz**（3V）/ **42 nV/√Hz**（5V） |
| 电流噪声密度         | **0.8 pA/√Hz**                       |
